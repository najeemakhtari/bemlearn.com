Full-Stack Roadmap

Our roadmap is designed for anyone, even with no programming experience. All you need is hard work and persistence. Let’s explore the concepts you’ll master on this journey.

What is web development?

Start from basic

In this section you will learn what is web development, we start from zero and you will learn the first steps of web development and what it this

The backbone of webpages

HTML

HTML (HyperText Markup Language) is the standard language for creating and structuring web pages. It defines the content and layout using elements and tags.

Style your webpages

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages, controlling the look and feel of elements. It defines colors, fonts, spacing, and positioning.

Control webpage behavior

JavaScript

JavaScript is a programming language that enables dynamic content and interactivity on web pages. It's used for client-side scripting in web development.

Control your webapp versions

Git + Github

Git is a version control system for tracking code changes. GitHub hosts Git repositories for collaboration and sharing.

Style faster in modern way

Tailwind CSS

Tailwind CSS is a utility-first framework that lets you style directly with predefined classes, simplifying design and customization.

Create user interfaces from components

React

React is a JavaScript library for building user interfaces using reusable components. It efficiently updates and renders components based on data changes.

TypeScript is JavaScript with syntax for types

TypeScript

TypeScript is a superset of JavaScript that adds optional static types for better error checking and code quality.

The react framework for web

Next

Next.js is a React framework that enables fast web apps with server-side rendering, static generation, and easy routing for better performance and SEO.

Make your webapp progresive

PWA

A PWA is a web app that works offline, loads fast, and feels like a native app using service workers and caching.

Run JavaScript everywhere

Node JS

Node.js is a server-side JavaScript runtime that enables fast, scalable web applications using non-blocking I/O.

Create database

MongoDB & mySQL

MongoDB is a NoSQL database that stores flexible, JSON-like documents. MySQL is a relational database that organizes data into structured tables.

Real-time, two-way communication made simple

WebSocket

WebSocket is a protocol enabling real-time, two-way communication between client and server over a single connection.

Scalable backend

Nest JS

NestJS is a TypeScript-based Node.js framework for building scalable, modular server-side applications.
